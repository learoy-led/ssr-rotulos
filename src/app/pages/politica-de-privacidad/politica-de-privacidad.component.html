Política de privacidad

La presente política de privacidad regula el tratamiento que hace {{contactDetails.shortName}} de los datos de los usuarios de este Sitio Web (en adelante, el Sitio Web).

Responsable del tratamiento de datos y delegado de protección de datos

  • Denominación social: {{contactDetails.longName}} (en adelante, {{contactDetails.shortName}})
  • Domicilio social y dirección postal: {{contactDetails.address1}} - {{contactDetails.address2}}
  • Dirección de correo electrónico: {{contactDetails.email}}
  • Código de Identificación Fiscal: {{contactDetails.cif}}

Para cualquier cuestión relacionada con protección de datos personales, puedes contactar con {{contactDetails.shortName}} en su dirección postal o en {{contactDetails.email}}.

Tipo de datos que tratamos

  • Datos sobre el uso que realizas de este Sitio Web.
  • Datos identificativos y de contacto
  • Información profesional

Dicha información es necesaria para cumplir con las finalidades que se describen en el siguiente apartado.

Finalidad del tratamiento de datos

La información recopilada en el Sitio Web únicamente será utilizada para mejorar la experiencia de los usuarios del Sitio Web y responder a sus consultas o informarles sobre nuestros productos y servicios.

Legitimazión para el tratamiento de datos

La base legal para el tratamiento de tus datos es tu consentimiento mediante la aceptación del uso de cookies o mediante el envío de una consulta a través de un formulario de contacto.

Plazo de conservación de los datos

Los datos personales se conservan durante el plazo que dura la relación informativa. Los datos de carácter no personal se conservan durante el período de conservación de cada cookie.

Cesión de datos a terceros

{{contactDetails.shortName}} no cede datos personales a terceros. Unicamente cede datos de carácter no personal a terceros mediante la aceptación del uso de cookies tal y como se informa en nuestra Política de cookies.

Derechos de protección de datos

Puedes ejercer tu derecho de acceso, rectificación, portabilidad, cancelación y limitación del tratamiento, así como oponerte al mismo enviando una solicitud por escrito a {{contactDetails.address1}} - {{contactDetails.address2}} o a {{contactDetails.address1}} - {{contactDetails.email}}

También tienes derecho a presentar una reclamación ante la Autoridad de Control: la Agencia Española de Protección de Datos (www.agpd.es).

Modificaciones en esta política

Es posible que actualicemos la Política de privacidad de nuestro Sitio Web periódicamente. Por ello, te recomendamos revisar esta política cada vez que accedas a nuestro Sitio Web con el objetivo de estar adecuadamente informado sobre cómo y para qué utilizamos tus datos.
